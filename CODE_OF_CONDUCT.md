# Code of Conduct

## Our Pledge

为了营造一个开放、友好、专业的社区环境，参与者应当保持尊重、克制与建设性沟通。

## Our Standards

积极行为包括：

- 以事实和问题为中心讨论技术内容
- 尊重不同经验背景的贡献者
- 接受合理反馈并持续改进
- 在文档、Issue、PR 中保持清晰表达

不可接受的行为包括：

- 人身攻击、侮辱、威胁或骚扰
- 恶意引战、刷屏、挑衅性言论
- 泄露他人隐私或敏感信息
- 故意发布恶意代码、漏洞利用或未脱敏凭据

## Enforcement Responsibilities

维护者有权删除不合适内容、关闭不合适讨论，或限制破坏性参与行为。

## Scope

本行为准则适用于：

- Issue
- Pull Request
- 代码评审
- 文档讨论
- 其他与项目相关的公开互动

## Reporting

如果你遇到不当行为，请联系维护者并提供必要上下文。
